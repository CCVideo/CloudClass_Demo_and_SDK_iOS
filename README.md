## 环境准备

- Xcode 8.2.1 及以上版本
- iOS 8.0 及以上版本


## 安装快速入门


下载完毕后进入./Demo/CCClassRoom `CCClassRoom.xcworkspace`，打开：

```shell
 $ open CCClassRoom.xcworkspace
```

在 Xcode 中编译运行快速入门，把快速入门安装到 iPhone 中。
